# Petal

from the word 'sociopetal'

## Quick Start
Install dependencies:
```bash
bun install
```
## Frontend dev server
```bash
bun run dev
```
opens at http://localhost:1420
## Full Tauri dev (desktop app + hot reload)
```bash
bun run tauri dev
```
## Production build
```bash
bun run tauri build
```
binaries in src-tauri/target/release/bundle/
